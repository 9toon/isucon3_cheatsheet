https://github.com/tmm1/stackprof

```ruby
# config.ru
require 'stackprof'
use StackProf::Middleware, enabled: true,
                           mode: :cpu,
                           interval: 1000,
                           save_every: 5
```

```
$ bundle stackprof tmp/stackprof-cpu-*.dump --text
```
